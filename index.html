01

Bias Bars: Gender-Based Review Analysis

Analyzing linguistic patterns in academic reviews using NLP

Project Summary

Conducted research at Stanford analyzing gender bias in professor reviews. Built comprehensive data pipelines in Python to process 50,000+ reviews and identify linguistic pattern disparities between male and female faculty evaluations. Developed an interactive Tkinter GUI visualization tool allowing real-time exploration of word frequency distributions, sentiment analysis metrics, and statistical significance testing. The analysis revealed significant differences in review language, with female professors receiving more personal criticism while male professors received more feedback on teaching methodologies.

Key Achievements

  • Processed and analyzed 50,000+ professor reviews using NLP techniques
  • Designed Python data pipeline with pandas for ETL and preprocessing
  • Built interactive Tkinter GUI with real-time visualization capabilities
  • Identified statistically significant gender bias patterns in language usage
  • Presented findings at Stanford summer research symposium

Tech Stack

02

Financial Investment vs. Performance in ACC Basketball

Predictive analysis of NCAA program funding and win outcomes

Project Summary

Built a predictive ML model using a decision tree regressor to analyze the correlation between NCAA ACC program funding and win percentage across 15 schools. Achieved an R² score of 0.78 forecasting 2023 season outcomes. Performed extensive data cleaning on historical dataset spanning 10 years, implemented feature engineering to extract meaningful predictors from raw budget data, and validated model performance using cross-validation techniques. The findings suggest a moderate but significant relationship between program investment and competitive performance.

Key Achievements

  • Built decision tree regressor model with 78% accuracy (R² = 0.78)
  • Processed 10 years of historical ACC basketball and budget data
  • Implemented feature engineering for budget-to-performance correlation analysis
  • Validated model predictions for 2023 season against actual outcomes
  • Identified key budget allocation strategies used by top-performing teams

2D Dungeon Crawler

Fully playable game with dynamic level progression and scoring

Project Summary
